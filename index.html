Sophos Installation

Follow these steps to install Sophos:

  1. Press Windows + R
  2. Type \\sinsfps1\downtimelogs\ and press Enter
  3. Locate and run the Sophos Installation file
Patch Checking

To verify installed patches, use the following method:

  1. Press Windows + R
  2. Type cmd and press Enter
  3. In the Command Prompt, run: wmic qfe list
Patch Checking 2.0

To check the Windows version:

  1. Press Windows + R
  2. Type winver and press Enter
