Sophos Installation

Follow these steps to install Sophos:

  1. Press Windows + R
  2. Type:
    \\sinsfps1\downtimelogs\
  3. Locate and run Sophos_Setup_Official_Inspiro.exe
Patch Checking

To verify installed patches:

  1. Press Windows + R
  2. Type:
    cmd
  3. Run:
    wmic qfe list
Patch Checking 2.0

To check Windows version:

  1. Press Windows + R
  2. Type:
    winver
Patching

Follow these steps for patching:

  1. Press Windows + R
  2. Type:
    \\sinsfps1\downtimelogs\jan
  3. Press Enter
  4. Run the patch installer
Check Drive Restriction

To check if the C: drive is restricted:

  1. Press Windows + R
  2. Type:
    C:
Enable/Disable USB Access

To toggle USB access:

  1. Press Windows + R
  2. Type:
    \\sinsfps1\downtimelogs\Shortcuts
  3. Run:
    _usboff
Installing MS Office 2016/2019

Steps to install MS Office 2016 or 2019 (Pro and Standard):

  1. Press Windows + R
  2. Type one of these:
    \\sinsfps1\downtimelogs\WAH-Apps\MS Office 2016 Pro and Standard
    \\sinsfps1\downtimelogs\WAH-Apps\Office 2019
  3. Press Enter
  4. Choose and install the desired version
Hostname Checking

To check the computer's hostname:

  1. Press Windows + R
  2. Type:
    cmd
  3. Run:
    hostname
