Sophos Installation

Follow these steps to install Sophos:

  1. Press Windows + R
  2. Type:
    \\sinsfps1\downtimelogs\
  3. Locate and run Sophos_Setup_Official_Inspiro.exe
Patch Checking

To verify installed patches:

  1. Press Windows + R
  2. Type:
    cmd
  3. Run:
    wmic qfe list
Enable/Disable USB Access

To toggle USB access:

  1. Press Windows + R
  2. Type:
    \\sinsfps1\downtimelogs\Shortcuts
  3. Run:
    _usboff
Checking Hostname

To check the hostname:

  1. Press Windows + R
  2. Type:
    cmd
  3. Run:
    hostname
